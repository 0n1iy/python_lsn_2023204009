## 四、被控对象模型

**双惯性环节串联**：
```
G(s) = 1/((T₁s+1)(T₂s+1))
```

### 参数配置
| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| T₁ | 1.0 | >0 | 第一惯性环节时间常数 |
| T₂ | 2.0 | >0 | 第二惯性环节时间常数 |
| 增益 | 1.0 | >0 | 被控对象总增益 |

### 信号流向
```
控制量u → [惯性环节1] → 中间变量 → [惯性环节2] → +干扰 → 输出y
                               ↓
                         内环反馈(串级)
```

---

## 五、反馈环节

**一阶低通滤波器**：
```
H(s) = 1/(Tcs + 1),  Tc = 0.1s
```

用于模拟传感器动态特性。

## 七、控制策略（5种，运行中可无缝切换）

### 7.1 普通PID控制（无限幅）
- 纯PID算法，无抗积分饱和
- 无输出限幅（教学演示用）
- 可观察积分饱和现象

### 7.2 单回路PID控制（带抗饱和）
- 位置式PID算法
- 条件积分抗饱和（误差方向有利于退出饱和时累积积分）
- 输出限幅 [-30, 30]

### 7.3 前馈+反馈控制
- 前馈补偿可测干扰
- 静态前馈增益：Gff = -Kd/Kp = -0.5
- 反馈部分使用PID

### 7.4 串级PID控制
- 外环：控制被控对象最终输出
- 内环：控制第一惯性环节输出（中间变量）
- 外环反馈：第二环节输出+干扰
- 内环反馈：第一环节输出

### 7.5 串级+前馈控制
- 串级结构基础上叠加前馈补偿
- 前馈作用于内环输出（最终控制量）

## 九、抗积分饱和策略

### 9.1 条件积分（Condition Integration）
```
if (output > U_MAX and error > 0) or (output < U_MIN and error < 0):
    暂停积分累积  # 会加剧饱和
else:
    正常累积积分   # 有利于退出饱和或正常调节
```

### 9.2 手动/自动切换
- 切换瞬间控制量保持不变（无扰动切换）
- 控制器积分自动跟踪当前输出值
- 确保切换回自动时无冲击

## 十、干扰信号

### 10.1 方波干扰
- **振幅**：可配置（默认 0）
- **持续时间**：可配置（秒）
- **作用位置**：叠加在被控对象输出端

### 10.2 操作逻辑
1. 设置振幅和持续时间
2. 点击"施加方波干扰"按钮
3. 按钮自动锁定，显示倒计时
4. 持续时间结束后按钮恢复

### 10.3 干扰与反馈
- 串级控制：外环反馈包含干扰（原始输出）
- 单回路控制：PV显示经过反馈环节的值

## 十三、异常处理机制

### 13.1 捕获范围
- 参数输入异常（非法值、越界）
- 计算异常（除零、溢出）
- 文件读写异常（权限、磁盘满）
- UI操作异常（空指针、越界）
- 网络/数据库异常

### 13.2 处理原则
1. **不闪退**：任何异常都不导致程序退出
2. **友好提示**：中文错误信息，用户可理解
3. **状态保持**：异常后程序状态可恢复
4. **日志记录**：错误信息写入日志文件

### 13.3 异常提示方式
- 弹窗提示（阻塞式，用户必须确认）
- 状态栏提示（非阻塞）
- 日志文件记录（详细堆栈信息）
