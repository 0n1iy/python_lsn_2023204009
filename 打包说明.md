# 打包说明

## 环境要求

- Python 3.10+
- Windows 10/11
- 依赖包已安装：`pip install -r requirements.txt`

## 一键打包

```bash
python build_exe.py
```

## 手动打包

```bash
pyinstaller PID仿真系统.spec
```

或者指定spec文件：

```bash
pyinstaller --onefile --windowed --name="PID温度控制仿真系统" src/main.py
```

## 输出

打包生成的文件：

```
dist/
└── PID温度控制仿真系统.exe    # 单文件可执行程序
```

## 注意事项

1. 首次打包时间较长（约2-5分钟），请耐心等待
2. 打包过程中可能会弹出Windows Defender警告，请允许
3. 如果报找不到pyqtgraph，请先执行：`pip install pyqtgraph`
4. 打包后的EXE文件较大（约80-150MB），包含完整的Python运行时和依赖库
5. 发布时需要在EXE同目录下创建 `data/` 文件夹（程序也会自动创建）

## 发布清单

将以下文件打包为ZIP发布：

```
发布包/
├── PID温度控制仿真系统.exe
└── 使用说明.txt
```

程序首次运行时会自动创建 `data/` 目录和默认用户数据。

## 常见问题

**Q: 打包后运行报错 "No module named 'xxx'"**
A: 在 `build_exe.py` 的 `pyinstaller_args` 中将该模块添加到 `--hidden-import`

**Q: 打包后界面是英文或样式丢失**
A: 确保 `assets/style.qss` 文件被正确包含（已在打包脚本中配置）

**Q: EXE太大怎么办？**
A: 可使用 `pip install pyinstaller` 后手动精简依赖，或在干净虚拟环境中打包
